# 机构专属知识库（待导入）

当前已有一份受控导入文件：[organization-knowledge.json](organization-knowledge.json)。它只保留了安全的沟通流程与资料状态，没有把原始价格、证书样本、联系方式或未复核的项目事实放入 AI 上下文。

将已审核的资料放入这个目录，或直接在对话中上传给我。推荐整理成下列字段：

| 内容类型 | 建议字段 |
| --- | --- |
| 招生方案 | 省份、年度/批次、路径、院校、专业、层次、资格条件、学习/考试要求、收费项目、退款规则、官方来源、有效期、复核人 |
| 话术 | 场景、客户意图、推荐说法、禁用说法、适用条件、审核人、版本日期 |
| 电销 SOP | 通话阶段、必问问题、转人工条件、允许的下一步、风险提示 |

不要在资料中保存身份证号、手机号、银行卡号、验证码、未脱敏录音或证件影像。对于每条可能变化的政策信息，请保留官方链接和有效期。

只有同时具备“官方来源、适用省份/批次、有效期、复核人”的记录，才应写入 `approvedFacts` 供 AI 对外使用；其余资料只能作为“需核验”的内部提醒。
